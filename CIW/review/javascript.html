4.2: Define JavaScript

  • Scripting language

  • Event driven
  • Platform independent

4.3: Comparing JavaScript to other languages in web development

  • JScript
    • Microsoft's implementation of JavaScript
    • Differences between JavaScript and JScript caused problems
  • ECMA
    • European Computer Manufactures Association
    • Approved JavaScript as an international standard in 1977
    • Tried to promote this standard as "ECMAScript"
  • VBScript
    • Visual Basic Script
    • Microsoft's original web scripting language
    • Based on the Visual Basic programing language
  • Java
    • Has no relation to JavaScript
    • Was the "hot new language" at the time that JavaScript was created, and JavaScript was named after it as a marketing ploy.

4.4: Embedding JavaScript into HTML5 documents

  • JavaScript code goes inside of script tags
  • Script tags can go either in the head or the body
  • You can link to external javascript files by using the src attribute in a script tag
  • If a script tag links to a file with the src attribute, all code within the tag is ignored
