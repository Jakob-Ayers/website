  • Basic Selectors
    • *: selects all elements
    • Type selector: Selects all elements of a given element name
    • Class selector: Selects all elements of a given class
    • ID selector: Selects an element with a given ID.
    • Attribute Selector: Selects elements based on their attributes
  • You can group selectors by separating them with commas.
  • Combinators
    • Decedent Combinator: Whitespace, selects all elements that are a decedent of a given selector.
    • Child Combinator: >, selects children of a given selector.
    • Sibling Combinator: ~, selects elements that have the same parent of a given selector
    • Adjacent Sibling Combinator: +, selects siblings of a selector that immediately follow a given selector.
    • Column Combinator: ||, selects elements that belong to a column, has no browser support why do i need to know this?
  • Pseudo
    • Pseudo class: element:checked, element:hover, ect., selects elements based on state information
    • Pseudo element: ::first-line ::before, selects entities not included in html.
