Types

  • Number
    • are numbers
    • fractional numbers (floats) are written with a . (ie 3.5)
    • Very big/small numbers are written in scientific form (ie 2.998e8)
    • Since there are only 64 bits to represent each number, math with floats is less precises.
    • Special Numbers
      • Infinity - its infinity
      • -Infinity - negative infinity
      • NaN - not a number
        • Despite the name, NaN is a number (by type)
        • NaN is the only js value that is not equal to itself
        • NaN is the result of doing illegal operations, such as Infinity-Infinity, 0/0, or adding a string (that doesn't have numbers) and a number
  • String
    • Characters n stuff
    • strings can be defined with "" '' or ``
    • \ can be used for escape characters
    • Encoded with unicode
    • While you cant do math with strings, you can use the + operator to add 2 strings together
    • Template Literals
      • strings defined with backticks (``) are called template literals
      • these strings can span multiple lines
      • things written inside of ${} will be computed (when within a string defined with backticks)
  • Boolean
    • True or False
    • Comparison will result in a boolean value (ie 1 > 2 will return false)

Operators

  • Arithmetic
    • + - addition
    • - - subtraction
    • / - division
    • * - multiplication
    • parentheses can be used to change the order of operations, like in normal math
  • Unary Operators
    • Defined with words, not symbols
    • console.log - prints to console
    • typeof - returns the type of a given value
  • Comparison
    • == - equals
    • != - not equal
    • < - greater than
    • > - less than
  • Logical Operators
    • && - and (will result in true if both values are true)
    • || - or (produces true if either value is true)
    • ! - not (flips the value given to it)
    • The last value
      • Ternary, not binary
